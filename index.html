Tharuni

Tharuni is presented here as a simple, elegant nameplate for a personal web presence: warm, direct, and easy to expand. This page is just a first placeholder, meant to hold space for a fuller story, portfolio, profile, or collection of notes later.

Identity Layer A lightweight static front end with room for biography, links, images, writing, and future modular content blocks.
Deployment Surface Structured for GitHub Pages now, with a clean path to CDN-backed hosting on CloudFront or any other static delivery platform later.
Content Model Placeholder copy today, editable HTML tomorrow. The goal is low friction iteration without introducing unnecessary stack overhead.
Technical Posture Minimal operational complexity, stable markup, and a straightforward deploy pipeline with enough headroom for future enhancement.

This is intentionally simple jargon-forward starter content. You can replace the copy, visuals, layout, or structure later without changing the basic deployment setup.
